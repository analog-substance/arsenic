## Suggested Installation

Arsenic is intended to be checked out along side other similarly purposed tools in an `opt/` directory.



## Requirements

- curl
- ncat
- nmap
- awk
- sed
- grep
- figlet
- exploitdb (searchsploit)

## Recommendations

- xenon
- baymax
